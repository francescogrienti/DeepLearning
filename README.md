# Deep Learning Exercises - University of Milan

This repository contains a collection of exercises and assignments developed for the **Deep Learning** course, held by **Prof. Stefano Carrazza** at the **Università degli Studi di Milano** (UNIMI).

The course covers fundamental and advanced topics in Deep Learning, ranging from classical machine learning concepts to modern neural network architectures.

## Repository Structure

The repository is organized by lecture, each containing Python implementations of the assigned exercises:

- **Lecture 1: Python and OOP Refresh**
  - Focus on Object-Oriented Programming principles and basic numerical computing.
- **Lecture 2: NumPy Fundamentals**
  - Exercises on data manipulation and basic operations using NumPy.
- **Lecture 3: Multi-Layer Perceptrons (MLP)**
  - Manual implementation and random initialization of MLP architectures using TensorFlow.
- **Lecture 4: Regression Models**
  - Building and training regression models using Keras and TensorFlow.
- **Lecture 5: Hyperparameter Optimization**
  - Exploring Bayesian Optimization techniques using the `hyperopt` library.
- **Lecture 6: Classification and Data Analysis**
  - Working with the Iris dataset, Pandas integration, and TensorFlow classification models.
- **Lecture 7: Convolutional Neural Networks (CNN)**
  - Implementation of CNN architectures for image classification tasks (e.g., CIFAR-10).

## Requirements

The exercises are implemented in Python and primarily rely on the following libraries:
- `numpy`
- `tensorflow`
- `keras`
- `pandas`
- `matplotlib`
- `hyperopt`
- `scikit-learn`
